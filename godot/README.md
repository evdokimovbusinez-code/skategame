# SK8TOWN — Godot 4.7

Open `project.godot` in Godot 4.7 and run `scenes/main.tscn` (F6/F5).

This is a code-native playable port: the district, interiors, props, characters and UI
are constructed by GDScript at runtime, so no import step or external asset pack is
required. Start at the central DIY park, ride to the Burnout shop, grind the rails and
talk to the local crew.

## Controls

- `WASD`: ride / steer / walk
- `Space`: ollie / jump / leave grind
- `F`: kickflip while airborne
- `Tab`: skate / walk
- `E`: interact
- `R`: reset
- `Esc`: pause
- Mouse: camera
